# SAE1.01-02_Brouillimg
## Programme de chiffrage et déchiffrage d'image en Java
## Projet de groupe à l'IUT Nord Franche-Comté

Pour exécuter ce programme, veuillez vous assurer d'avoir Java d'installé et d'avoir compilé les fichiers.
Pour toute l'exécution, veuillez utiliser un terminal.
Pour compiler : javac *.java
Pour brouiller une image : java Brouillimg <chemin d'accès de l'image> scramble <clé à utiliser (entier entre 0 et 32767 inclus)> <image de sortie (si non précisé : out.png)>
Pour débrouiller une image : java Brouillimg <chemin d'accès de l'image> unscramble <méthode à utiliser (euclide / pearson)> <vitesse d'execution (slow / fast)> <image de sortie (si non précisé : out.png)>
